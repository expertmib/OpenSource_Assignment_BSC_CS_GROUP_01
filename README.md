# Student Information Management System

## Degree Program
Bachelor of Science in Computer Science

## Group Number
Group 01

## Project Description
This project is a Student Information Management System developed using PHP.
The system is designed for primary and secondary schools in Tanzania.

### Features
1. Register Students
2. Display Student Records
3. Search Student by Registration Number

### Technologies Used
- PHP
- MySQL
- HTML
- CSS
- Git
- GitHub
